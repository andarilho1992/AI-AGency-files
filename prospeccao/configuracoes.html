Configure sua API Key do YouTube para começar

🔑 Como obter a API Key (gratuita, 5 min)
A YouTube Data API v3 é gratuita. Você tem 10.000 unidades por dia — suficiente para analisar centenas de comentários diariamente.
1

Acesse o Google Cloud Console

Vá em console.cloud.google.com e faça login com sua conta Google.

2

Crie um projeto

Clique em "Selecionar projeto" → "Novo projeto" → dê um nome (ex: prospeccao-andarilho) → Criar.

3

Ative a YouTube Data API v3

No menu lateral: "APIs e serviços" → "Biblioteca" → busque YouTube Data API v3 → clique → "Ativar".

4

Crie a credencial (API Key)

"APIs e serviços" → "Credenciais" → "+ Criar credenciais" → "Chave de API". A chave aparece na tela — copie ela.

5

Cole a chave abaixo e salve

Pronto. A chave fica salva só no seu navegador, nunca vai para a internet.
