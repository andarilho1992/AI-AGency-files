01. O ESQUELETO

O navegador lê de cima para baixo. Use tags semânticas para o Google te amar:

  • <main>: Conteúdo principal.
  • <section>: Blocos de assunto.
  • <article>: Post ou Card isolado.
  • <aside>: Barra lateral ou info extra.

02. AUTO LAYOUT (FLEX)

No CSS, display: flex; é o seu melhor amigo. Ele alinha tudo.

justify-content: space-between;

Isso espalha os itens como o "Space Between" do Figma.

03. COMPONENTIZAÇÃO

Classes (.): Estilos que se repetem (ex: botões, cards).

IDs (#): Elementos únicos (ex: Menu de navegação).

.card-premium { border-radius: 8px; }

04. HACKEANDO O LAYOUT

Para mudar algo rápido, procure no CSS:

  • Padding: Respiro interno.
  • Margin: Distância externa.
  • Opacity: 0 a 1 (transparência).
  • Display: none; Esconde o elemento.
