Dicas rápidas
💡
Dados ficam no dispositivo
O sistema salva tudo no próprio celular/computador. Não precisa de internet para usar — só para abrir pela primeira vez. A sincronização entre dispositivos está no roadmap.
⚠️
Cada aparelho tem seus dados
Venda feita no computador não aparece no celular automaticamente (ainda). Use sempre o mesmo dispositivo por turno para evitar confusão.
💾
Faça backup regularmente
Em Configurações → Exportar dados. Salva tudo em um arquivo JSON. Se precisar trocar de dispositivo, importe o arquivo no novo aparelho.
📱
Instale como app no celular
Android: Chrome → menu ⋮ → "Adicionar à tela inicial". iPhone: Safari → compartilhar → "Adicionar à tela de início". O ícone aparece na tela como qualquer app.
