## O que foi feito

### Hospedagem S2 — Checklist diário
- Tab "Rotina do dia" no prontuário do hóspede
- Alimentação: checkboxes manhã/tarde/noite com horário previsto (07:30/13:00/19:00), registro de plantonista + hora real
- Medicações: geradas automaticamente dos horários cadastrados no prontuário, cada slot com checkbox + quem administrou + hora real
- Atividades: Passeio / Recreação / Socialização (toggle)
- Obs do dia: texto livre, salvo automaticamente
- Badge vermelho "Atrasado" (30min de tolerância após horário previsto)
- Navegação de datas: ver dias anteriores
- Plantonista: select populado de D.funcionarios, nome registrado em cada ação
- Dados em `plano.checklist[dateISO]` no localStorage

### Internação Veterinária — Módulo completo
- Nav item "Internação" com badge de internados ativos
- Lista com tabs: Internados agora / Todos / Com alta
- Modal "Internar animal": tutor + pet + data entrada + previsão alta + diagnóstico + médico + obs
- Prontuário — aba Admissão: dados clínicos editáveis + medicações com **via de administração** (VO/SC/IM/IV) + botão "Dar alta"
- Prontuário — aba Rotina: checklist de medicações + alimentação (Aceitou/Recusou) + evolução do dia (texto livre)
- Relatório tutor: abre nova janela com PDF imprimível — cabeçalho do pet shop, dados do animal, tabela de medicações com status, alimentação, evolução do dia
- Dados em `D.internacoes[]` (array separado de `D.planos`)

### Schema DB
- `D.internacoes` adicionado a `loadDB()` e `emptyDB()`
- `internacoes[].checklist[dateISO]` com medicações (geradas da prescrição), alimentacao {aceitou, obs}, evolucao
