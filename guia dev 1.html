LVL 1: DESIGNER TO DEV

1. O INVENTÁRIO (Tags & Classes)

Pense nas Tags como os tipos de itens (Espada, Poção) e as Classes como os encantamentos (Fogo, Gelo).

<button class="btn-fire">Atacar</button>

2. MAPA DO TESOURO (Inspecionar)

Aperte F12 no Chrome. É o seu "mapa do mundo". Clique na seta de seleção e aponte para qualquer elemento para ver as entranhas do código.

Não tenha medo de deletar coisas no Inspecionar. Nada que você faz ali é permanente. É o seu playground!

3. POÇÕES DE CURA (CSS Rápido)

  • Padding: Espaço interno (enchimento).
  • Margin: Espaço externo (distância).
  • Display Flex: O "Auto Layout" do código.

4. QUEST: O DESAFIO DO CARD

Tente recriar um card de personagem usando apenas div para o box e span para o HP.

.card { border: 4px solid #fff; padding: 10px; }
.hp-bar { background: red; height: 10px; }
