O que está sendo construído

Pets Go não é um sistema para o seu pet shop. É um produto para o mercado pet.

Desde o início, a ideia nunca foi construir um sistema que funciona só para você. A ideia é construir o melhor software para gestão de pet shops do Brasil — e usar o seu negócio como o primeiro laboratório vivo.

Isso muda tudo: o que você está financiando não é uma ferramenta interna. Você está co-fundando um produto que vai ser vendido para centenas de pet shops. Suas 4 filiais são a prova de que funciona.

Por que o Pets Go ganha do mercado

O que nenhum sistema existente oferece

🤖

Bot de WhatsApp com inteligência artificial — de verdade

Os sistemas existentes (SimplesVet, Petshop Control) têm "integração com WhatsApp Web" — o funcionário ainda precisa digitar. Pets Go terá um bot que responde sozinho: confirma agendamento, avisa quando o pet está pronto, manda lembrete de vacina, sem ninguém precisar tocar no celular. Isso não existe no mercado para pet shop pequeno.

🏪

Feito para pet shop, não para clínica veterinária

Os principais sistemas do mercado foram pensados para clínicas veterinárias — prontuário, internação, receita médica. Pet shop tem um fluxo diferente: banho, tosa, agendamento por profissional, recompra de ração. Pets Go foi construído em cima do mapeamento real do seu negócio.

📱

Reativação automática de clientes inativos

Um cliente que não aparece há 60 dias recebe uma mensagem automática no WhatsApp. Sem funcionário ligando. Sem planilha manual. O sistema identifica, manda a mensagem e registra a resposta. Isso já existe no Pets Go — nenhum concorrente faz isso automaticamente para pet shop.

🔄

Preparado para escalar para múltiplas filiais

O sistema está sendo construído para que um dono com 4, 10, 20 filiais veja tudo em um único painel. Cada filial separada, mas visão consolidada. Isso é exatamente o que o Junior precisa hoje — e o que outros donos de rede vão pagar para ter.

Transparência total

O que ainda falta ser honesto

Um parceiro de verdade fala a verdade. Aqui está o que ainda está sendo construído:

O produto ainda está em desenvolvimento

O CRM básico está no ar. Agenda, PDV, Financeiro completo e WhatsApp bot ativo ainda estão sendo construídos. Isso leva tempo e segue o roadmap definido juntos.

O bot de WhatsApp aguarda um investimento de infraestrutura

Para o bot funcionar de verdade, precisamos de um servidor dedicado (~R$25/mês) e a conexão com a Evolution API. Esse é o próximo passo técnico depois de estabilizar o CRM atual.

Somos uma equipe pequena — e isso tem vantagens e limitações

Hoje é Guilherme desenvolvendo com suporte de inteligência artificial avançada. Isso significa velocidade de desenvolvimento acima do normal, mas também significa que não somos uma empresa de 50 pessoas com suporte 24/7. Conforme o produto cresce, a estrutura cresce junto.

O que precisa acontecer agora

Os próximos passos concretos

1 — Técnico (esta semana)

Resolver a comunicação entre o CRM e os agentes de IA

Há um problema técnico pendente desde 16/04 que impede o CRM de conversar com os agentes inteligentes. Resolver isso desbloqueia as automações.

2 — Dados (quando possível)

Para o sistema funcionar com dados reais, preciso da exportação dos clientes do sistema atual (Petshop Control). Isso transforma o sistema de teste em sistema operacional.

3 — Infraestrutura (próximo mês)

Ativar o bot de WhatsApp

Com o servidor estabilizado e a conexão configurada, o bot entra em operação. A partir daí, o sistema começa a trabalhar sozinho no atendimento.

4 — Estratégia (em paralelo)

Definir o acordo de longo prazo

Se o Junior vai ser o canal de expansão para outros pet shops, precisamos definir como isso funciona — modelo de indicação, participação nos resultados, condições para as próximas filiais que entram.
