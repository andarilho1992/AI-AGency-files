LEVEL UP: DESIGNER -> FRONT-END ARCHITECT

01. O BOX MODEL

Tudo no HTML é um quadrado. Entender isso é 90% do trabalho.

margin: fora
border: a linha
padding: dentro

02. ALINHAMENTO DINÂMICO

Quer centralizar algo? Use o "Combo de Ouro":

display: flex;
justify-content: center;
align-items: center;

03. MODO HACKER (F12)

O navegador é seu editor em tempo real. Altere cores no painel Styles para testar.

> Tente mudar meu 'font-size' no Chrome!

04. RESPONSIVIDADE

No CSS, use % ou rem em vez de px fixos para que o site respire.

width: 100%;
max-width: 1200px;
