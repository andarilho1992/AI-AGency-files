Mês 2+ — replicação

Prospecta outras creches pets com o case do Junior. Vende o sistema pronto. Cada novo cliente = R$ 200/mês de margem com custo fixo já pago.

PostgreSQL: +R$ 28/mês quando 3+ clientes

Bottom line

O cartão do Junior cobre tudo que é necessário para construir e operar o sistema dele — incluindo as ferramentas de desenvolvimento. Seu custo real nesse mês é zero. A partir do mês 2, a mensalidade de R$ 400 do Junior paga o custo operacional com R$ 200 de margem sobrando. Cada novo cliente replica essa margem com custo incremental baixo.
