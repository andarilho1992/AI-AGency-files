Zona Agência

Automação & Sistemas

Ferramentas que constroem o produto que você vende — bots, CRM, fluxos e agentes autônomos. Aqui entra o dinheiro recorrente.

n8n
O orquestrador — conecta tudo
Grátis (self) / $20/mês

É o sistema nervoso central da agência. Pega uma informação de qualquer lugar (WhatsApp, formulário, e-mail, Google Sheets) e decide o que fazer com ela — sem você precisar estar online. Você cria fluxos visuais conectando blocos: "quando X acontecer → faça Y → depois Z".

Exemplo real — Bot de atendimento

Cliente manda "quero orçamento" no WhatsApp → n8n recebe via Z-API → manda pra Claude classificar a intenção → se for lead quente, registra no CRM e avisa você → se for dúvida simples, Claude responde automaticamente.

Como começar
1Cria conta em app.n8n.cloud (14 dias grátis)
2Cria workflow: "Webhook recebe dados → salva no Google Sheets"
3Testa mandando uma requisição pelo browser
4Adiciona Claude e Z-API quando o básico funcionar
workflows visuaissem código280+ integraçõesself-hosted gratuito
Claude API
O cérebro — entende e decide
~R$2/1000 msgs

É o Claude que você usa agora, mas via API — chamado diretamente pelos seus sistemas, não pelo chat. O modelo Haiku é o ideal para bots: barato, rápido, suficientemente inteligente pra qualificar lead, responder dúvida, gerar proposta. Você define a personalidade no system prompt e ele age como um atendente treinado.

Exemplo real — Bot do cliente pet shop

System prompt define: "Você é a assistente da Creche Pet. Responda sobre check-in, preços e horários. Se o tutor perguntar sobre vagas, peça nome e contato e registre no sistema." — Claude faz exatamente isso, no tom certo, 24h por dia.

Como começar
1Acessa platform.anthropic.com e cria API key
2No n8n, adiciona node "HTTP Request" apontando pra API
3Escreve o system prompt do personagem do bot
4Testa mandando mensagens e ajusta o prompt

Claude Managed Agents
Agentes autônomos — trabalham sozinhos por horas
$0.08/hora runtime

Lançado em abril 2026. É Claude com autonomia total para executar tarefas longas sem você supervisionar: abre browser, lê arquivos, faz buscas, escreve código, chama APIs — e não para até terminar a tarefa. Diferente da API normal que responde uma mensagem por vez, aqui o agente trabalha por horas seguidas com memória de sessão.

Exemplo real — Prospecção autônoma

Todo domingo às 20h, um agente acorda, busca 30 restaurantes novos no Google Maps da cidade alvo, verifica quais não têm site ou têm site lento, cria fichas no seu CRM com nome/contato/problema encontrado, e te manda um resumo no WhatsApp. Você acorda segunda com leads prontos pra abordar.

Como começar (depois do Claude API básico)
1Mesma API key do Claude — só muda o header da requisição
2Adiciona: anthropic-beta: managed-agents-2026-04-01
3Define o agente: modelo + system prompt + ferramentas permitidas
4Inicia sessão e monitora via eventos em tempo real

Z-API
A ponte — WhatsApp ↔ seus sistemas
~R$150/mês

WhatsApp não tem API oficial acessível pra pequenos negócios. Z-API é o meio-campo: você conecta um número de WhatsApp a ele (via QR Code), e a partir daí consegue receber e enviar mensagens via código. Sem ele, seu bot não existe. Com ele, qualquer mensagem que chega no WhatsApp do cliente pode ser processada pelo n8n e respondida pelo Claude.

Exemplo real — Qualificação de lead

Lead manda "oi" no WhatsApp do pet shop → Z-API captura → manda pro n8n → n8n manda pro Claude → Claude responde com boas-vindas + pergunta sobre o pet → lead responde → Claude coleta dados → n8n salva no CRM → dono recebe notificação só quando o lead está qualificado.

Como começar
1Cria conta em z-api.io
2Cria uma instância e escaneia QR Code com o número do cliente
3Copia o token e a URL do webhook
4Cola no n8n como trigger do workflow
WhatsApp não-oficialwebhookenvio e recebimentopor instância
Make (Integromat)
Alternativa visual ao n8n — mais simples
Grátis até 1k ops/mês

Mesmo conceito do n8n, interface mais amigável, hospedagem no cloud (não precisa servidor). Ideal quando o cliente quer algo simples e você não quer configurar infraestrutura. O free tier é suficiente para clientes pequenos — formulário de contato → CRM → e-mail automático.

Exemplo real — Onboarding de novo cliente

Cliente assina o serviço → Make pega os dados do formulário → cria pasta no Google Drive → manda e-mail de boas-vindas personalizado → cria linha no CRM → agenda reunião de kickoff no Google Calendar → manda lembrete 24h antes. Tudo automático, sem você tocar em nada.

Como começar
1Cria conta em make.com
2Escolhe um template pronto (tem centenas)
3Adapta pras suas ferramentas (Google, e-mail, CRM)
4Testa com dados reais e ativa
no-codecloud nativo1000+ appstemplates prontos
Google Sheets + Apps Script
CRM gratuito com automação nativa
Grátis

Planilha que vira CRM completo com o Apps Script (JavaScript que roda dentro do Google). Você cria colunas para cliente, status, data, valor — e o Script envia e-mail automático quando status muda, cria relatório semanal, alerta quando proposta está vencendo. É o CRM mínimo viável antes de vender o CRM customizado pra cliente.

Exemplo real — Seu próprio pipeline de vendas

Coluna "Status" com: Prospectado / Contato feito / Demo enviada / Proposta enviada / Fechado / Perdido. Quando muda pra "Proposta enviada", Apps Script agenda follow-up automático em D+1, D+3 e D+7. Você nunca esquece de um lead.

Como começar
1Cria planilha no Google Sheets com colunas de CRM
2Extensões → Apps Script → cola o script de automação
3Define gatilho: "quando linha muda → executa função"
4Testa com um lead real

Presença Digital & SEO

Ferramentas para construir, otimizar e automatizar a presença online dos clientes. Cada site entregue gera mensalidade de manutenção.

Claude — Copywriter de Sites
Escreve o site inteiro em minutos
Incluso no plano

Você fornece as informações do negócio (nicho, público, diferencial, localização) e o Claude escreve headline, subtítulo, seção "sobre", lista de serviços, depoimentos fictícios para placeholder, meta description e título SEO. O que levaria 4 horas de escrita vira 20 minutos de revisão.

Exemplo real — Site de eletricista

Prompt: "Escreva o copy de uma landing page para João Silva, eletricista em Florianópolis, atende residencial e comercial, 15 anos de experiência, destaque para rapidez e preço justo. Tom direto, sem enrolação." — Em 30 segundos você tem o texto completo do site.

Como usar no fluxo
1Faz uma call de 15min com o cliente e anota os dados
2Cola no Claude com o prompt de copywriter
3Claude gera headline, corpo e meta tags
4Você revisa e encaixa no HTML

Google Search Console
Raio-X de como o Google enxerga o site
Grátis

Mostra em quais buscas o site aparece, em qual posição, quantos cliques recebe, se tem erros de indexação e se o Google consegue ler as páginas. É a ferramenta obrigatória de qualquer serviço de SEO — sem ela você está gerenciando às cegas. Você entrega o acesso ao cliente como parte do pacote.

Exemplo real — Auditoria grátis de prospecção

Você pede acesso ao Search Console do lead (ou verifica por fora com outras ferramentas) e mostra: "Seu site aparece na posição 34 pra 'eletricista florianópolis'. Ninguém chega na página 4. Posso colocar você no top 5 em 60 dias." — Esse dado fecha o diagnóstico.

Como começar
1search.google.com/search-console → adiciona o site
2Verifica propriedade (tag no HTML ou DNS)
3Submete o sitemap.xml
4Aguarda 48h e começa a ver dados

Zona Canal

YouTube & Instagram

Ferramentas que multiplicam um único vídeo em múltiplos conteúdos — sem gravar mais, sem perder a naturalidade.

OpusClip
Um YouTube vira 5 Reels automaticamente
Grátis 90min/mês

Você faz upload do vídeo longo ou cola o link do YouTube. A IA assiste o vídeo inteiro, identifica os momentos mais relevantes e poderosos, corta em clipes verticais de 30–90 segundos, adiciona legenda animada e reformata o enquadramento pra vertical. O que levaria 3 horas de edição acontece em 10 minutos.

Exemplo real — Semana de conteúdo em 1 upload

Você grava o YouTube de 12 minutos no domingo (a jornada da semana). Sobe no OpusClip. Ele corta: o momento em que você fala do cliente recusando a proposta (Reel segunda), quando você mostra o bot funcionando (quarta), quando revela quanto entrou de dinheiro (quinta). Você tem 5 Reels sem gravar mais nada.

Como começar
1opus.pro → cria conta grátis
2Cola o link do YouTube ou faz upload do arquivo
3Escolhe o estilo de legenda e aspecto (9:16)
4Baixa os clipes e revisa rapidamente antes de postar
corte automáticolegenda animadareformato verticalhighlight detection
Descript
Edita vídeo como se fosse um texto
Grátis limitado / $12/mês

Transcreve o vídeo automaticamente e exibe o texto lado a lado com o vídeo. Para cortar um trecho, você apaga o texto — o vídeo corta junto. Quer remover "ãhn" e silêncios? Um clique. Isso é revolucionário pra quem filma natural sem roteiro — você edita conversando com o texto, não arrastando linha do tempo.

Exemplo real — Estilo Casey Neistat

Você grava 40 minutos de vídeo bruto durante a semana — café, rua, computador, conversa com lead. Sobe no Descript. Ele transcreve. Você lê o texto e deleta as partes chatas. Em 45 minutos de edição você tem um YouTube de 12 minutos que parece documentário.

Como começar
1descript.com → cria conta e instala o app
2Importa o vídeo — transcrição automática em PT
3Lê o texto, seleciona e deleta o que não quer
4Exporta o vídeo editado

VidIQ
SEO do YouTube — aparece na busca certa
Grátis básico

Extensão do Chrome que aparece ao lado de qualquer vídeo do YouTube mostrando: quantas views por hora, quais palavras-chave estão trazendo tráfego, qual a dificuldade de ranquear e o que os concorrentes estão usando. Você escreve títulos e tags com base em dados reais, não achismo.

Exemplo real — Título que ranqueia

Você quer postar sobre "como fazer bot de WhatsApp". VidIQ mostra: esse termo tem 12k buscas/mês e dificuldade média. Sugere variação: "bot WhatsApp grátis para pequenos negócios" — 8k buscas, dificuldade baixa. Você usa o segundo e aparece no top 10 mais rápido.

Como começar
1vidiq.com → instala extensão no Chrome
2Conecta com sua conta do YouTube
3Antes de postar, pesquisa o título no buscador do YouTube
4Usa as sugestões de palavras-chave no título e descrição

Presentes nas três zonas

Ferramentas que aparecem em qualquer projeto. Quanto mais cedo dominar, mais velocidade você tem.

Claude (chat)
O assistente que não dorme
Plano atual

Já é seu principal parceiro de trabalho. O que ainda pode expandir: criar um arquivo de contexto fixo (quem você é, seus projetos, preços, tom de voz) e colar no início de cada conversa importante. Isso evita reexplicar tudo e mantém consistência. Use pra: proposta, copy, roteiro, diagnóstico de cliente, código, estratégia.

Exemplo real — Proposta em 15 minutos

Cola o contexto fixo + dados do cliente + serviço desejado. Claude gera proposta completa com problema, solução, etapas, preços e prazo. Você revisa, personaliza 20% e manda. O que levava 2 horas vira 15 minutos.

Próximo nível
1Cria um arquivo .txt com seu contexto fixo
2Sempre cola no início de conversas estratégicas
3Cria prompts salvos pra tarefas repetidas (proposta, follow-up, roteiro)
4Quando precisar de mais poder → migra pra API
