Instalar como app no celular
Pets Go no celular — sem precisar de internet
O sistema funciona instalado como app. Depois de instalar, abre igual ao WhatsApp — sem precisar digitar link. Funciona offline também.
Android (Chrome)
1. Abra o link do sistema no Chrome
2. Toque no menu ⋮ (canto superior direito)
3. Toque em "Adicionar à tela inicial"
4. Confirme → ícone aparece na tela
iPhone (Safari)
1. Abra o link do sistema no Safari
2. Toque no botão compartilhar (quadrado com seta)
3. Role e toque "Adicionar à tela de início"
4. Confirme → ícone aparece na tela
Funciona sem internet: depois de instalado, o sistema fica salvo no dispositivo. Você consegue agendar, registrar diárias e ver o caixa mesmo sem sinal.
Usar no computador
Computador da loja — como funciona
Abre no navegador como qualquer site. Pode ser instalado como app. Funciona offline depois da primeira carga.
1
Abrir: Chrome ou Edge → acesse o link do sistema. Na primeira vez precisa de internet. Depois de carregar uma vez, funciona offline normalmente — o sistema fica salvo no próprio computador.
2
Instalar como app (opcional): na barra de endereço do Chrome/Edge aparece um ícone de instalar ( ou ). Clique → confirme → o Pets Go abre em janela própria, sem barra do navegador, igual a um programa instalado.
✓
Sem internet na loja? Sem problema. Depois de instalado, tudo funciona normalmente — agenda, caixa, planos. Os dados ficam salvos no dispositivo.
Atenção — dados por dispositivo: hoje cada aparelho tem seu próprio banco de dados. Se você registrar uma venda no computador, ela não aparece automaticamente no celular. A sincronização entre dispositivos está em desenvolvimento (próxima versão).
Próxima versão — Sync entre dispositivos
Dados em todos os aparelhos ao mesmo tempo. Funciona offline e sincroniza quando a internet voltar.

Como vai funcionar: cada aparelho continua salvando os dados localmente. Quando tiver internet, sincroniza automaticamente com a nuvem. Se a internet cair no meio do turno, o sistema não para — só sincroniza depois.
💡
Na prática: venda feita no caixa do computador → aparece no celular em segundos. Agendamento criado no celular → aparece no computador da recepção. Tudo em tempo real.
Essa funcionalidade está planejada e será ativada assim que você aprovar as funcionalidades atuais. Nenhuma mudança no visual ou no uso — só os dados passam a se comunicar entre dispositivos.

Pagar direto na agenda

Histórico de diárias
Botão 📋 em cada plano mostra todos os dias usados e o valor de cada diária. Responda na hora quando o cliente perguntar.

Diária retroativa com data
Esqueceu de lançar ontem? Sem problema — escolha a data ao usar a diária. O sistema avisa se já tem lançamento naquele dia.

Após concluir e pagar, o sistema pergunta "Agendar próxima visita?". Escolha +7, +15 ou +30 dias — agendamento criado com um clique, mesmo pet e serviço.

Como o sistema funciona
Tudo fica no menu lateral — no celular aparece como ☰. Cada módulo cuida de uma área do pet shop. Números em vermelho ou amarelo no menu são alertas em tempo real: planos críticos, pets em check-in, etc.

Planos & Pacotes
Planos & Pacotes Atualizado
Pacotes de diárias para creche, hotel e serviços recorrentes. Cada diária usada lança receita no caixa automaticamente.
1
Criar plano: + Novo Plano → preencha os campos:
Grupo
Creche · Hospedagem · Banho/Tosa · Estética · Outro
Modalidade
Mensal · Trimestral · Semestral · Anual · Avulso
Frequência (Creche)
2× a 6× por semana — aparece só para o grupo Creche
Modo de abate
Manual (creche/pacotes) ou Automático (hospedagem)
N
Usar diária com data: Usar diária → escolha a data → confirme. Funciona retroativamente. Se já houver lançamento naquele dia, o sistema avisa antes de criar outro.
N
Ver histórico: clique no botão em qualquer plano — vê todos os dias usados com data e valor.
2
Barra de progresso: verde = tranquilo · amarelo = atenção · vermelho = crítico (menos de 3 dias).
3
Renovar: quando estiver acabando, Renovar adiciona mais crédito sem criar plano novo.
4
Filtros: Ativos ⚠️ Críticos Todos Vencidos Cancelados
N
Cancelar plano: o botão ✕ cancela o plano e remove o lançamento financeiro da contratação. As diárias já usadas continuam no histórico.
O número em amarelo na sidebar mostra quantos planos têm menos de 3 diárias. Verifique toda manhã.
Clientes & Pets
Clientes & Pets
Cadastro completo de tutores e animais. CPF, endereço, peso do pet e histórico de serviços.
1
Novo cliente: + Novo Cliente → nome, WhatsApp, CPF, endereço → adicione o pet com raça, porte e peso em kg.
2
Porte do pet — preencha sempre: o sistema usa o porte para calcular o preço automaticamente ao agendar.
Pequeno — até 10kg Médio — 11 a 20kg Grande — 21 a 30kg Extra G — acima de 30kg
3
Planos do cliente: clique em 🎟 Planos no card do tutor para ver, criar e dar baixa em diárias diretamente.
4
Histórico do pet: clique no pet para ver todos os serviços realizados e os valores cobrados.
Caixa & Financeiro
PDV / Caixa
Para vendas no balcão sem agendamento. Serviço com agendamento → conclua pela Agenda (entra no caixa automático).
1
Serviço avulso: "Serviço" → escolha → informe pagamento → Registrar Venda → recibo abre automático.
2
Produto: "Produto" → escolha o item → preço preenche sozinho → confirme. Estoque descontado automaticamente.
3
Vender plano: "Plano / Pacote" → cliente → valor total e diária → confirmar. Plano criado automaticamente em Planos & Pacotes.
Recibo automático: toda venda gera um recibo com três botões — NFS-e para emitir nota fiscal, Imprimir / PDF para enviar ao cliente, e Fechar. Para reemitir: Financeiro → ícone ao lado do lançamento.
Financeiro
Receitas entram automaticamente pela Agenda e Planos. Dois níveis de acesso.
Movimento
Acesso livre. Receitas do dia. Qualquer funcionário pode ver.
Gerencial
Com senha. Despesas, contas a pagar/receber e saldo real do mês.
!
Configurar senha: Configurações → "Senha Gerencial" → definir e salvar antes de usar.
