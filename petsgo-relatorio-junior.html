Seção 01

Resumo Executivo

O sistema Pets Go Control está completo e em produção. Todos os 11 módulos foram entregues e estão funcionando — da agenda ao financeiro, passando por hospedagem e internação veterinária. O acesso é por link direto no navegador, sem instalação, e funciona tanto no computador quanto no celular. O sistema roda offline e armazena todos os dados localmente, sem custo de servidor.

Seção 03

Segurança e Qualidade Técnica

Na versão mais recente (07/05/2026) foram aplicadas melhorias técnicas de segurança e performance, sem nenhuma mudança visível para o usuário final:

🛡️
Proteção de dados
Todos os textos digitados — nomes, observações, diagnósticos — são verificados antes de serem exibidos, prevenindo corrupção de informações no banco local.
🔐
Senha gerencial protegida
A senha da área financeira agora é armazenada com criptografia (hash SHA-256), não mais em texto simples. Ninguém consegue ler a senha diretamente nos dados salvos.
⏱️
Bloqueio automático
A área financeira trava automaticamente após 15 minutos sem uso, protegendo o caixa quando o computador fica sem supervisão.
⚡
Performance aprimorada
O banco de dados local é lido uma única vez por interação (cache inteligente), tornando o sistema mais ágil — especialmente em fichas com muito histórico.
✔️
Validações de negócio
O sistema impede dois agendamentos no mesmo horário para o mesmo profissional. Planos de hospedagem exigem que o pet esteja devidamente cadastrado antes.
🗑️
Proteção contra exclusão acidental
Para apagar todos os dados do sistema, é necessário digitar manualmente a palavra "APAGAR" — evitando perdas por clique acidental.
Seção 04

Como o Sistema Funciona

  • Roda 100% no navegador — sem servidor próprio, sem mensalidade de infraestrutura. O sistema é acessado direto pelo Chrome, Edge ou Firefox.
  • Dados salvos localmente — todas as informações ficam no computador (localStorage do navegador). Funciona normalmente mesmo sem internet.
  • Um único arquivo HTML — toda a lógica, visual e dados em um só lugar. Atualizar o sistema é simples e fazer backup é trivial.
  • Link de acesso permanente: andarilho-digital.pages.dev/petsgo-crm-light.html
  • Compatível com Chrome, Edge e Firefox em desktop e mobile. Recomendado: Chrome no computador para melhor experiência.
